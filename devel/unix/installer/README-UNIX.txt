
------------------------------------------------------------------------------------------

1. FIRST TIME USE : INSTALLATION

 1. Decompress the .tar.gz archive somewhere 

------------------------------------------------------------------------------------------

2. NORMAL USE : STARTING THE APPLICATION

  1. open a Terminal (or Console)
  2. go in the tinasoft desktop directory
  3. if it don't work, type "cd /Applications/Tinasoft"
  4. type "./tinasoft.sh"  (or "sh tinasoft.sh" if it fails)

  A browser should now open the "http://localhost:8888" website
  
  If it don't, start you browser (Firefox or Chrome) manually, then go to the following address (the ":8888" is mandatory):

    http://localhost:8888 


------------------------------------------------------------------------------------------

3. TROUBLESHOOTING

 If you don't see your whitelists or source-files, just refresh the page (this is not automatic yet).
 
 Safari is recommended
 
 You need a Spreadsheet editor
 
 When you close your Browser or the Webpage, this won't stop the application and running computations.
 
 To really stop all computations, you need to click on "Server Shutdown"
