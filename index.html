Data set : a directed and weighted graph database

Produce graphs

  • choose period(s) within a dataset and a whitelist

  • select relationships between documents and keyphrases

  • update the data set with new edges

  • optionnally export the entire graph to gexf file and publish it online with Tinaweb or explore it with Gephi

Explore graphs

  • Navigating in the network
    • Translation: Click anywhere and drag the mouse
    • Zoom: Use the mouse scroll, or double-click

  • Node selection and related informations (right panel)
    • Click on a a node to append it to the current selection
    • Click on a selected node to remove it from the current selection
    • Double-click on a node to switch to it's local neighbourhood
    • Click on the close button, in the right panel, to empty the selection
    • Change the cursor size, in the top toolbar, to select multiple nodes at the same time

  • Graph filtering
    • The top toolbar contains node's and edge's weight filters
    • Spatialization automatically adapts to the filtered graph
